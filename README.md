# Projet Réseau Embarqué

## A propos

Développement d'une couche liaison simplifiée en implémentation d'une stratégie d'accès au médium. L'accès sera contrôlé par une architecture de type time-triggered (partage du temps global entre les émetteurs).

### Fonctionnalités implémentées

* Capacitée d'envoyer des messages avec le SX1272
* Capacitée de recevoir des messages avec le SX1272

### Fonctionnalités futures

* Time Triggered Architecture

## Installation

* Atollic : Installer [Atollic True Studio](https://atollic.com/truestudio/).
* PuTTY : Installer [PuTTY](https://putty.org).

## Matériel

* STM32F072RB
* SX1272
